# Data Playground

This folder demonstrates JSON, YAML, CSV, and .env usage.

## Files
- **samples.json** — structured sample data used by applications.
- **samples.csv** — tabular version of the same sample data.
- **config.yaml** — human-friendly configuration file.
- **.env.example** — environment variables used for local development.

## Why these formats?

### JSON
Used for API payloads and machine communication.

### YAML
Used for configuration because it is human-friendly.

### CSV
Used for spreadsheets and dataset storage.

### .env
Used for secrets and environment-specific configuration.
